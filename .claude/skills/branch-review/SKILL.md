# Branch Review

You will be reviewing all changes on this branch compared to the main branch. Follow the steps in order. Finish each step before moving on to the next.

---

## Step 1: Understand the Changes

Form a clear understanding of the intent of the changes.

1. Read the full diff of this branch vs main
2. Read any changed/added files in full (not just the diff) to understand context
3. Answer these questions for yourself:
   - What problem is being solved?
   - What is the approach?
   - What are the key design choices?

**Output:** Write a "## Understanding of Change" section to the user. This is the ONLY output for this step.

When complete, read and follow `.claude/skills/branch-review/branch-review-step-2.md`.
